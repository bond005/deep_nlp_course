# Курс по обработке естественного языка (NLP)


1. **Семинар №1** - Введение в обработку и классификацию текстов
   - Модели классификации и предсказания
   - Основы информационного поиска

2. **Семинар №2** — Векторизация текста
   - Статистические подходы
   - Bag of Words (мешок слов)
   - Методы извлечения признаков

3. **Семинар №3** — Сверточные нейронные сети
   - Архитектура CNN и операция свертки
   - Применение к обработке текстов

4. **Семинар №4** — Обработка последовательностей
   - Рекуррентные нейронные сети (RNN)
   - Архитектура Transformer
   - Механизмы внимания (Attention)

5. **Семинар №5** — Метрики качества
   - Метрики для задач классификации
   - Метрики для задач регрессии
   - Методы оценки моделей 

6. **Семинар №6** — Методы PEFT
   - Parameter-Efficient Fine-Tuning

7. **Семинар №7** — Быстрый инференс LLM
   - Методы оптимизации инференса
   - От Multi-head Attention к Group Query Attention
   - Снижение вычислительной сложности

8. **Семинар №8** — Квантование моделей
   - Методы снижения потребления памяти
   - Преобразование типов данных (float32 → float16/bfloat16)
   - Стратегии квантования для LLM
